## Aufgabe 1: 

**Multi-Channel-Chatbot für eine Buchhandlung**

Eine Buchhandlung möchte einen einfachen Chatbot-Prototypen entwickeln, der Kundenanfragen über WhatsApp und einen weiteren Kanal, zum Beispiel einen Web-Chat, beantworten kann. Der Chatbot soll einfache Fragen zu Öffnungszeiten, Buchinformationen und Bestellstatus bearbeiten.

a) Entwickle ein Python-Skript, das eine einfache WhatsApp-Business-API-Anbindung vorbereitet. Die Klasse soll eine Nachricht an eine Kundennummer senden können. Verwende Platzhalter für Access Token, Phone Number ID und Kundennummer.

b) Implementiere zusätzlich einen einfachen zweiten Kanal, zum Beispiel einen Web-Chat, der Kundennachrichten simuliert verarbeitet.

c) Implementiere eine Funktion, die auf Anfragen zu Öffnungszeiten, Buchinformationen und Bestellstatus basierend auf einem vordefinierten Datensatz antwortet. Der Datensatz kann direkt im Skript als Dictionary gespeichert werden.

d) Teste den Chatbot mit mehreren Beispielnachrichten.

e) Begründe kurz, warum du einen gemeinsamen Datensatz und eine gemeinsame Antwortfunktion für beide Kanäle verwendest.
